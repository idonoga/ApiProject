UsersDB Restful API

Pass Data with JSON files.

Fields:

User-ID -> INT
User-Name -> STRING
User-Email -> STRING
             User-Password -> STRING            

API options:

Name of the feature Description Method
/ListUsers Listing All The Users in DB  GET 
/ListUser/<userID> List a Specific User in the DB GET
/AddUser Adding a user  POST
/ModifyUser Modify existing user  PUT
/DeleteUser/<UserID> Delete existing user DELETE

 
Enjoy!
